## Locate the my.cnf configuration file.

````bash
$ mysqladmin --help
````

~~~bash
Default options are read from the following files in the given order:
/etc/my.cnf
/etc/mysql/my.cnf
/usr/etc/my.cnf
~/.my.cnf
~~~
