Comparer deux chaines en ignorant les accents. PHP

by Arnaud Tiérant | 2019-11-05 | development

  1. Convertir les accents en leurs homologues non accentués, puis comparer les chaînes.
  2. Résultat

Convertir les accents en leurs homologues non accentués, puis comparer les chaînes.

$removeAccents = function (string $string) {
    return strtolower(
        trim(
            preg_replace(
                '~[^0-9a-z]+~i',
                '-',
                preg_replace(
                    '~&([a-z]{1,2})(acute|cedil|circ|grave|lig|orn|ring|slash|th|tilde|uml);~i',
                    '$1',
                    htmlentities($string, ENT_QUOTES, 'UTF-8')
                )
            ),
            ' '
        )
    );
};
$sentences = [
    'depot',
    'Dépôt',
    'Depôt',
    'Dépot',
    'Depot',
    'credit',
    'Dépôt et créDit',
    'Depôt et credit',
    'Depot et~   CRÉDIT',
];
var_dump(array_map(
    function ($sentence) use ($removeAccents) {
        return $removeAccents($sentence);
    },
    $sentences
));
